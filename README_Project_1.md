# Project 1 - Task List Program

This program is a simple command-line task list app.

What it does:
- Lets the user add tasks
- Shows all saved tasks
- Exits when the user chooses the exit option

How it works:
- The program keeps tasks in a list in memory
- It shows a menu with three choices
- When you add a task, it is stored in the list
- When you view tasks, it prints each task with a number

How to run:
- Open a terminal in this folder
- Run `python Project_1.py`

Notes:
- Tasks are not saved after the program closes
- This is a basic example of a menu-driven Python program

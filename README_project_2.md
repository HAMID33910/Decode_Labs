# Project 2 - Expense Tracker

This program is a simple command-line expense tracker.

What it does:
- Accepts expense amounts from the user
- Stops when the user enters `q`
- Shows the full list of expenses
- Calculates the total amount spent

How it works:
- The program stores each entered expense in a list
- Each value is converted to a decimal number
- After quitting, it prints the expenses and their sum

How to run:
- Open a terminal in this folder
- Run `python project_2.py`

Notes:
- The program expects numeric input for expenses
- It does not save data after the program closes

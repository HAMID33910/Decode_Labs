# Project 3 - Password Generator

This program generates a random password of a user-defined length.

What it does:
- Asks the user for a password length
- Builds a password using letters and numbers
- Prints the generated password

How it works:
- The program uses Python's `random` module to pick characters
- It uses letters and digits from the `string` module
- It repeats until the password reaches the chosen length

How to run:
- Open a terminal in this folder
- Run `python project_3.py`

Notes:
- The password only contains letters and numbers
- It does not include symbols

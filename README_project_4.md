# Project 4 - General Knowledge Quiz

This program is a simple command-line quiz game.

What it does:
- Asks the user for their name
- Presents a set of multiple-choice general knowledge questions
- Validates each answer and gives immediate feedback
- Shows the final score and grade at the end

How it works:
- The program stores questions, options, and correct answers in a list of dictionaries
- It loops through each question and prints the choices
- It checks user input for valid answers (A, B, C, D)
- It counts correct answers and calculates a final percentage

How to run:
- Open a terminal in this folder
- Run `python project_4.py`

Notes:
- The quiz does not save scores or progress after it ends
- Only the answers A, B, C, or D are accepted
